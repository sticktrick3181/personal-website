### How?

This website is built using [Next.js](https://nextjs.org) which is an extremely popular framework for [React.js](https://reactjs.org) and also my go-to framework for building any kind of websites. I highly recommend it. Lot of best practices built right into the framework by default.

The blogs are authored in Markdown using [MDX](https://mdxjs.com/). I could have set up a [Ghost](https://ghost.org/) backend to improve the authoring experience and to have a clear separation of content and UI but I want to avoid servers right now (or paying for hosted version of ghost).

For the UI, I have used [Tailwind CSS](https://tailwindcss.com/) which really helps me "design at the speed of thought".

The last bit of fun I had was to implement a simple page views counter. For that I am using [Hasura](https://hasura.io) as the backend. And that's pretty much it in terms of technologies used. You can find the code for the entire website [here](https://github.com/iNishant/personal-website).
